Cross Platform Video Capture Library
====================================

Video Capture is a cross platform library to capture video frames from 
capture devices. This library uses modern SDKs/APIs for Mac, Linux and
Windows.  This library is tested and developed on Win 8.1, Mac 10.9
and Arch Linux (latest Linux).
